# Get Stack Overflow posts App

## Stack and noteworthy dependencies

* Package manager - Yarn
* Main - React Native
* State management - mobx
* Network - Axios

## Setup

1. Install packages  
```
$ yarn install
```

2. Generate pods  
```
$ cd ios && pod install
```

## Running the project  
* iOS  
```
$ yarn ios
```
* Android  
```
$ yarn android
```
